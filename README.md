# Tajam
Tajam is a parallax business landing page.

### Features
1. Carousel
1. Parallax Scrolling
1. Themed Overlay
1. Gallery
1. Testimonials
1. Revealing Footer.
1. Code Comments
1. Responsive
1. Easily Add/Remove/Copy Sections
